# VCOLOR.JS
vcolor e um pacote simples para adicionar cores nos seus projetos de terminal node

## documentação
use a função console.log(VCOLOR("TEXTO ", "COR")) 
para add um texto

## cores do  VCOLOR
 preto: 
  vermelho: 
  verde:
  amarelo: 
  azul: 
  magenta: 
  ciano: 
  branco: 
  reset: reseta a cor
